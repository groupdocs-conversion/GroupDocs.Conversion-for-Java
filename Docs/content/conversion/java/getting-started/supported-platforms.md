## Supported Java Versions

GroupDocs.Conversion for Java supports Java versions that listed below. For more platform support please check GroupDocs.Conversion for .NET.

*   J2SE 7.0 (1.7)
*   J2SE 8.0 (1.8)
*   or above (*for example Java 10*)
